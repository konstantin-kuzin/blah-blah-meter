# Time Shit

Небольшое клиентское веб-приложение для оценки стоимости встречи по составу участников и длительности.

Пользователь указывает количество участников, их должности и оклады и на странице в реальном времени считается стоимость встречи.

![Скриншот приложения](assets/screenshot.png)

## Возможности

- расчёт стоимости встречи в реальном времени
- отображение стоимости одной минуты встречи
- учёт длительности встречи с возможностью вручную задать время старта
- учёт встречи по сегментам (раундам) с отдельной фиксацией длительности и стоимости каждого завершённого сегмента
- настройка заголовка страницы
- настройка списка должностей: название, оклад, добавление и удаление
- автоматическая сортировка по окладу от меньшего к большему
- переключение светлой и тёмной темы
- сохранение настроек и темы в `localStorage`
- опционально (beta): **распознавание речи** с выводом транскрипта поверх страницы

## Как запустить

Так как проект статический, достаточно открыть [index.html](/index.html) в браузере или [https://timeshit-counter.vercel.app/](https://timeshit-counter.vercel.app/)


Для режима **Groq Whisper** нужен доступ к маршруту `/api/groq` (например деплой на Vercel или локальный запуск в терминале командой: 
```text 
node start.cjs
```
Подробности — в [docs/PROJECT.md](docs/PROJECT.md).

## Как пользоваться

1. На основной странице укажите количество участников кнопками `+` и `−`.
2. При первом добавлении участника таймер встречи стартует автоматически.
3. Клик на `Продолжительность встречи` - изменить время начала.
4. Если встречу нужно разбить на отдельные части, используйте кнопку `Раунд`: текущий сегмент будет закрыт, а новый начнётся сразу после него. В списке под таймером отображаются только завершённые сегменты с их длительностью и стоимостью, а на кнопке показывается стоимость активного сегмента.
5. Через иконку настроек в правом верхнем углу можно:
   - изменить заголовок
   - изменить, добавить и удалить должности участников
   - на вкладке **Speech Recognition** выбрать движок: встроенный **Web Speech API** или **Groq Whisper** (при необходимости — ключ API и дополнительные опции)
6. Кнопка **микрофона** в тулбаре включает и выключает запись; **Shift+клик** по ней при выключенном микрофоне очищает текст транскрипта.

## Распознавание речи

Транскрипт показывается полупрозрачным слоем поверх интерфейса. В настройках можно переключиться между распознаванием в браузере (без локального сервера) и транскрипцией через **Groq** (нужен ключ API и запуск локального сервера). При желании доступны постобработка текста через LLM и перевод — они тоже идут через Groq.

## Логика расчёта

Стоимость минуты для каждого грейда считается по формуле:

```text
ratePerMinute = (salary * 1.5) / (160 * 60)
```

Где:
- `salary` — месячный оклад
- `1.5` — коэффициент стоимости грубо учитыающий налоги и отчисления в страховые фонды
- `160` — число рабочих часов в месяце

Общая стоимость встречи считается как сумма стоимости минут всех выбранных участников, умноженная на прошедшее время встречи.

Если использовать сегменты, приложение дополнительно ведёт внутреннюю разбивку общего таймера:

- первый сегмент начинается с момента старта встречи или с вручную заданного времени начала
- нажатие на кнопку `Раунд` завершает текущий сегмент и сразу открывает следующий
- в ленте раундов показываются только завершённые сегменты
- стоимость активного сегмента считается в реальном времени и показывается на кнопке `Раунд`
- при изменении времени начала встречи или при сбросе встречи сегменты пересоздаются заново
